System Architecture Prototypes

Exploring next-generation concepts in hardware management, storage, security, and HCI.

Operating Systems & Hardware Management

Predictive Thermal Management Kernel Module

The Concept

Moving beyond reactive cooling, this OS-level module uses a lightweight machine learning model to predict thermal spikes before they happen based on application launch patterns and active process queues.

How it Works

It reads system calls to anticipate heavy computational loads (e.g., initiating a video render). It then pre-emptively ramps up cooling or schedules background tasks to low-power cores before the thermal threshold is breached.

Potential Use Case: Extending battery life and preventing thermal throttling in ultra-thin laptops or high-density server racks.
Data Storage & Information Retrieval

Context-Aware Semantic File System

The Concept

Replacing traditional hierarchical folders with a relational organization system. It uses local AI processing to automatically scan, sort, and tag files based on their content, context, and relationships.

How it Works

Instead of manual folder sorting, the user simply saves a file. The system tags it with metadata (e.g., [Project X], [Invoice]). Users retrieve files by querying natural language rather than digging through directory trees.

Potential Use Case: Enterprise knowledge management and automated document organization for personal productivity.
System Security & Access Control

Continuous Biometric Authenticator

The Concept

Rather than authenticating only at login, this prototype continuously verifies the user's identity in the background using behavioral biometrics to ensure a zero-trust local environment.

How it Works

A background service analyzes keystroke dynamics and mouse movement patterns. If an anomaly is detected in the physical interaction style, the system instantly locks the session or drops privileges.

Potential Use Case: High-security environments where physical device hijacking or shoulder-surfing is a significant threat.
Distributed Systems & Networking

Decentralized Edge-Caching Mesh

The Concept

Allows devices on the same local network to act as a distributed, peer-to-peer cache, rather than every device pulling identical updates or media from a remote cloud server.

How it Works

When one PC downloads a large update, the system breaks the file into encrypted chunks stored across idle storage on the local network. Subsequent devices pull the update via gigabit local Wi-Fi rather than the internet.

Potential Use Case: Reducing bandwidth consumption and speeding up deployment times in corporate offices or bandwidth-constrained networks.
Human-Computer Interaction (HCI)

Spatial-Mapping Virtual Interface

The Concept

A system that turns any flat physical surface into a customizable input device without requiring a touch screen or expensive AR headsets.

How it Works

Using a downward-facing webcam, the system maps a desk space. Users define specific zones (e.g., "tapping this corner skips a track"). Computer vision tracks hand interactions within those boundaries and translates them into system macros.

Potential Use Case: Accessibility tools for users with limited mobility, or highly customized, invisible macro pads for productivity.
